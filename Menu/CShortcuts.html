Shortcuts
The Shortcuts screen allows you to manage your shortcuts. Each character has 20 shortcuts although only 10 can be displayed on the Shortcut Bar at anyone time due to size limitations.

There are a lot of different types of shortcut you can make, you can make shortcuts to screens, spells, actions, inventory items and crafting plans for example, these allow you to simply press a button on your Shortcut Bar or use the Alt + Number key shortcut to execute that shortcut and do whatever it should do.

If you shortcut an inventory item, then when you press the button it will do whatever clicking the Use button on the Inventory screen would normally do, if you shortcut a crafting plan then when you click the button it will load the crafting screen with that plan and let you skip the process of choosing it.

These shortcuts are a great way to speed up some processes and to customise the game to your own personal tastes and requirements.

The Shortcut screen has 20 buttons at the top of it, these represent your 20 Shortcut slots, clicking on one will let you edit the shortcut or add a new shortcut, after a shortcut is edited or added the Shortcut Bar on the right of the main screen will update with the changed details.

You can change the Image/Symbol and Text displayed on the Shortcut Bar via this screen as well.
