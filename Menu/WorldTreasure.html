Treasure
The Treasure screen can be used to find plan locations of treasure in the current realm, it will only show treasure on this map if you have the treasure map in your inventory.

If you click on the marked map you will be teleported to the location of the treasure, from there you need to dig up the treasure.



Digging up Treasure
To dig up treasure you need the Mining skill, this can be bought from an NPC in Knotwood Valley.

You need a Spade this can be bought from a general store, it may be listed under 1handed weapons.

Goto the tile with the treasure on it, equip the spade and click the Dig button ...
