All
The All screen is used to view your Backpack and Sellbag together in one screen, items in your Sellbag are displayed in red with a (s) prefixing them, all other items are in your Backpack.

You can use the buttons at the bottom to access different Inventory screens and the buttons on the right of the screen to perform certain actions with the selected item.

The bottom buttons do the following:

Equip
Shows the Equipment screen, where you can equip items.

Dismantle
Shows the Dismantle screen, where you can dismantle items.

Drop
Shows the Drop screen, where you can drop items.

Identify
Identify any unidentified items in your inventory that you have the skill to identify.

Stack
Stack any items in your inventory that can be stacked.

Queue
Shows the Queue screen, where you can manage your queued items.

Price
Shows the Price screen, where you can price goods for sale on your vendors or through the marketeer system.



The buttons on the right do the following:

Use
Try to use the selected item, if an item has no use it will tell you so when you click this button. If the item does have a use then it will perform that function.

Info
Shows the Information box for the selected item, useful for getting a full description of the item including stats and equip requirements etc ...

Equip
Equip the selected item, if there is already an equipped item in the slot then it will be unequipped and this item equipped instead.

Unequip
Unequip the selected item. The item is returned to your inventory.

Identify
Identify the selected unidentified item if you have enough skill.

Disguise
Disguise the selected item, which makes it unidentified.

Drop
Drop the selected item, some items can not be dropepd due to flag setting, you should use the Flags screen to change those.

Link
Sends a link containing information about that item to chat, this only works with ingame chat, players can then click on the link to view the information on that item.

Rename
Some items such as bought bags can be renamed using the Rename button on this screen, this allows you to call your bag something useful so you know what is in it.
