Resources
The Resources screen allows you to locate resources that you can gather with one of the gathering skills.

Resources spawn in a set location so the map will hardly ever change, it will not show a depleted resource but resources respawn after a few minutes in the same spot.



The Resource Map shows all of the resources that you have the skill to gather, you can change the type of resource you wish to show by clicking on the image of the resource in question in the right side of the screen.

This screen only shows a part of the current realm so you will need to use the Arrow buttons to move the viewed area box around the realm, as your skill gets higher it may be necessary for you to travel further across the realm to find resources for it.

You can click on the listed resource to goto that spot on the map and from there you just need to click on the resource node to beging the gathering process.

Pet Gathering Skill
If you have a pet and wish to use its gathering skill for the resource map, there is a yellow button labelled "pet" on this screen, clicking this will let you choose a pet, and then that pet will be used.
