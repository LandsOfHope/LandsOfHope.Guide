Shoutbox
The shoutbox is the default chat client there are generally a lot of people in the shoutbox.

Shoutbox Commands
CommandDescription
/nick new nameAllows you to change nicknames.

Example:
/nick David
/join channelAllows you to join the specified channel.

Example:
/join Trollsgrove
/me emoteAllows you to emotes.

Example:
/me dribbles
/q questionAllows you to send a question to chat, These questions appear differently than if you just typed a question.

Example:
/q How do I regenerate my Health ?
/a answerAllows you to send an answer to chat, These answers appear differently than if you just typed the answer.

Example:
/a Goto World > Rest
/ignore chat nickAllows you to ignore someone in the shoutbox. The ignore list is not saved outside of your game session, logging out and back in will clear it for example. This is a toggle command so if someone is ignored and you send the command again it will unignore them. Users can only be ignored if they have at least one line of chat visible in the shoutbox at the time.

Example:
/ignore David
/clearignoreAllows you to clear your temporary ignore list. The list is not saved anyway but it may become necessary to clear the list when you add some people to it you want to quickly remove.

Example:
/clearignore
/voteSends a voting link to chat, allowing people to click the link to vote for the game on mpogd.com, it may be considered spam if you click it too much.

Example:
/vote
/link urlSends a link to a website to chat, these appear differently than if you just typed the link and stand out a little more.

Example:
/link http://www.landsofhope.com
/infoDisplays your account id in the active channel, this can only be seen by you.

Example:
/info


Founder Commands
CommandDescription
/chadd channelCreates and registers a channel, making you a founder. Which enables you to use the following commands..

Example:
/chadd Myzan
/chdel channelUnregisters and deletes a channel.

Example:
/chdel Myzan
/chban nicknameBans the entered nickname from chat, this only works if that nickname is present in chat at the time.

Example:
/chban Bob
/chunban accountidUnbans an account, when you ban someone using chban it bans the entire account that means all characters. To unban you have to enter the account id of the person you banned.

Example:
/chunban 12323
/chuser accountidAdds the specified account to the list of users allowed to access this channel. It can be used in conjunction with a chmode command to only allow certain people to access the channel.

Example:
/chuser 12121
/chvoice accountidAdds the specified account to the list of users allowed to have moderator permissions in this channel. Moderators have the power to ban and unban users but can not use Founder commands.

Example:
/chvoice 12121
/chuserdel accountidRemoves the specified account from the list of users for this channel, this works to remove basic users, moderators and banned accounts.

Example:
/chuserdel 12121
/clearAllows you to clear the channel, this is useful if someone is spamming or you want to erase something that was in chat.

Example:
/clear
/chdesc descSets an optional description for the channel.

Example:
/chdesc The world of Myzan !
/chmode MODELets you change the channel mode ... more information on this below.
/chowner accountidChanges the owner of the channel you are currently in (and founder of) to the given account, useful if you are going away or need to give the channel to someone else.

Example:
/chowner 112123

Channel Modes
The following modes are used with the /chmode command and only work for Founders.
ModeDescription
/chmode NONEDefault Mode, in this mode anyone can type/access this channel.

Example:
/chmode NONE
/chmode USERUser Mode, this is when only people who have been added to the user list using either /chuser or /chvoice can type in this channel, others may still access the channel but they can only read it and not write to it.

Example:
/chmode USER
/chmode GUILDGuild Mode, only members of your guild can access the channel. In addition anyone added using /chuser or /chvoice can access the channel too.

Example:
/chmode GUILD
/chmode SERVERServer Mode, only people on the same server as the Founder (or those added using /chuser and /chvoice) can type in this channel.

Example:
/chmode SERVER
/chmode ALLEGIANCEAllegiance Mode, only people in the same allegiance as the Founder (or those added using /chuser and /chvoice) can type in this channel. If the founder changes allegiance then he/she will need to type /chmode ALLEGIANCE to have the new allegiance value updated.

Example:
/chmode ALLEGIANCE


The general chat rules apply to the Shoutbox, and at all times you are bound by the TOS you agreed to when making a game account.

If you abuse the shoutbox you may end up being banned from it, or in the very worst cases getting your account banned!
