Geology
The GEOLOGY gathering skill alllows you to gather raw resources. To gather this type of resource you must first find a gem or magmite resource node, this can be done with the World > Resources screen.

Resources gathered with this skill are added to your Inventory > Queue screen.

When you have found a geology resource node, click on it to be shown the gathering screen.



The gathering screen can then be used to take the resources from this node, each time you gather the amount of resources accessible at this node goes down 1, eventually when it reaches 0 you need to move to another node.

A depleted node may take several minutes to respawn, use the Inventory > Queue screen to check on the status of your items.
