Exploration
The Exploration screen displays all of the unexplored buildings in the realm that you have the skill to explore.

You need to have the Exploration skill to use this screen, the higher your skill the more buildings you have the potential to explore and the easier it becomes to raise the skill.

When you explore a building no one else can explore that building for up to an hour, this means if someone else is on and raising exploration it can be tough for others to do so but this makes exploration fun !.

Some buildings are shared across both servers, meaning if a player explores the building on one server its visible on the other, however there are a lot of server specific practice huts which can be explored.

Note: If a building has been explored it will not show up on this screen, until it reverts back to unexplored.



The screen lists all of the buildings you can explore, use the Arrow buttons to move the viewed area around the current realm as there may be some buildings you can explore in other parts of the map.

Click on a building when it is listed to teleport to that tile, from there you can use the Explore button to explore.
