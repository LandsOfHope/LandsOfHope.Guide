Exiting the Game
The recommended way to exit the game is to goto File > Logout, you will then need to click a button to confirm you wish to logout.

If you logout correctly in this manner then you will disappear from the game instantly, if you attempt to logout indirectly either through closing the browser or not playing the character it will still be visible for up to 15minutes at which point it will then automatically disappear.

If you are playing on a grief server and do not logout correctly you may find you get killed or stolen from between leaving the game and the 15minutes it takes for you to disappear.
