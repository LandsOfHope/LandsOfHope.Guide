Interface
The Interface is intended to be a very simple one, the game itself is very complex but we wanted the Interface to be very flexible and easy to use.

This is why the Interface is driven entirely by the mouse and why we have an extensive Menu system at the top of the screen.

I am sure many of you are used to games where there is a big bar of actions/functions/screens on the left of the window, I too have seen browser based games like and they are great for a small game with not many features but for a game with a lot of features a hierarchical menu structure is perfect.
