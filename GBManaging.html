Friends
There can only be one "owner" of a building, but there is a building friends list that you can add managers or administrators too allowing other characters or friends specific permissions to your building, that way you can setup complex buildings with independant room ownership or just a safe list of people to access your house.

If you lock your building then only players on your building friends list can enter it (unless your on a grief server in which case Lockpicking may allow someone to break).

To edit the current room friend list click the "Friends" button in the bottom bank of buttons on the main screen, this will then load the Friends screen where you can add a new friend, edit existing friends and change permissions etc ...



The buttons on this screen are fairly straight forward, they do the following: Save
Save the Admin and Manager check box changes.

Someone with Admin permissions can use many of the features such as rennovating, its a way to let a friend work on your building without them having the power to take the building from you.
Someone with Manager permissions can use the Property > Manage screens and have almost the same powers as the building owner.

Kick
Kicks that person from the room, only works if that person is inside the current room.

Ban
Ban the selected person from entering this room.

Unban
Removes a ban from someone if they were previously banned.

Delete
Delete the selected person from the Friend list.

Global
Make the room settings for the selected person apply to the entire building i.e. all rooms.

Global Delete
Delete the selected person from the Friend list for all rooms in this building.
