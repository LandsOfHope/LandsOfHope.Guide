Sets
The Sets screen is used to view a list of all of the sets you own a piece too.

A set is a collection of items that when all required pieces of a set are owned allows you to activate some special power or bonus.

There may be a lot of sets in the game, and collecting the pieces of one set could be hard work but there will almost always be a good reason for doing so.

The screen provides a place to view the pieces you are missing and also to view the pieces you have, as well as activating its special powers if you have gained all the required pieces.
