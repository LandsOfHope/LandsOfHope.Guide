Drop
The Drop screen can be used to select items that you wish to drop, a dropped item is deleted it does not appear on the ground and let you pick it up. Deleted items can not be restored and requests to do so will not be granted.

You can use click and drag to select multiple items on that screen, then use the middle drop button to drop multiple items at once or you can drop items individually.

Some items can not be dropped immediately, they may have a "No Drop" flag turned on you should turn off the No Drop flag using the Flags screen before dropping them, the No Drop flag is intended to prevent a player dropping an important item accidentally.
